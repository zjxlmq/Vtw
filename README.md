# 使用说明
```
来自异次元的订阅，能够让你在网上无限制浏览所有网站

每次不可用时，更新订阅

小火箭使用方法:

打开 Shadowrocket，点击右上角＋，类型选择 “Subscribe (订阅)”。
粘贴订阅链接至 URL 后，点击右上角 “完成”，Shadowrocket 将从订阅链接获取节点。
如果机场官网用户中心有显示“一键导入SSR”，那么一般在手机上直接登录官网然后点击此按钮也可以一键导入

v2ray使用方法:

然后我们打开v2rayNG，点击左边汉堡菜单，选择订阅分组设置，然后点击右上角＋号。

配置文件输入你想要起的名字，url填入刚刚复制的V2RAY订阅地址，点击√号即可保存

保存后回到v2ray app首页 点击右上，选择更新订阅。
```
